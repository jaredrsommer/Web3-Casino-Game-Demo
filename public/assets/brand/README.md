## Coreum Brand Colors

Following the official Coreum brand guidelines:

### Primary Colors
- **Primary Green**: `#25D695` (RGB: 37, 214, 149)
- **Dark**: `#101216` (RGB: 16, 18, 22)
- **Pure White**: `#FFFFFF` (RGB: 255, 255, 255)

### Typography
- **Primary Typeface**: Space Grotesk
- **Secondary Typeface**: Noto Sans

## Usage Guidelines

### Logo Usage
1. Maintain clear space around logos
2. Use the green wordmark for primary branding
3. The mascot can be used as an icon or secondary brand element

### Color Application
- Use Primary Green (#25D695) for:
  - Primary buttons and CTAs
  - Active states
  - Success indicators
  - Brand accents

- Use Dark (#101216) for:
  - Backgrounds
  - Text on light backgrounds
  - Navigation bars

### Asset Integration
All assets are optimized and ready for web use. Reference them using:

```tsx
// Logo wordmark
<img src="/assets/brand/COZY LOGO GREEN.png" alt="CozyCasino" />

// Mascot
<img src="/assets/brand/cozy.png" alt="CozyCasino Mascot" />

// Copied to image directory for easy access:
<img src="/assets/image/cozy-logo.png" alt="CozyCasino" />
<img src="/assets/image/cozy-mascot.png" alt="CozyCasino" />
```
